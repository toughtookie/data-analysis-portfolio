# 电商平台用户行为与品类运营分析

本项目基于淘宝用户行为数据，分析“双十二”前用户在平台上的浏览、加购、收藏与购买行为，挖掘高转化路径与品类特征，辅助平台优化运营策略。

## 📌 项目目标
- 分析用户在不同时间段的行为特征
- 识别高价值商品类别与转化路径
- 提出数据驱动的优化建议

## 🛠 技术栈
- Python（Pandas）进行数据清洗与分析
- MySQL 进行数据建模与查询
- Tableau 进行可视化展示

## 🔍 关键发现
- 用户活跃高峰集中在 10–15 点，收藏行为转化率低于加购  
- 浏览→加购→购买路径转化率最高  
- 品类决策时间与销量呈负相关
